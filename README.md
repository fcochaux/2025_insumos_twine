# Material audiovisual para hiperficción de Twine

Este repositorio contiene el material audiovisual utilizado en una hiperficción creada con **Twine**. Los archivos aquí almacenados (principalmente imágenes y otros recursos) se utilizan mediante enlaces directos dentro de la obra.

## Hiperficción

La obra completa puede leerse en el siguiente enlace:

https://tallervirtualdeescritores.com/wp-content/uploads/2025/07/Cantos-desde-el-humedal-Francisco-Chaux.html

## Nota

Los archivos de este repositorio funcionan como recursos externos para la hiperficción. Modificar la estructura de carpetas o eliminar archivos puede afectar la visualización de la obra publicada.
